# CPU・RAM

## 使用状況を見る

CPUカードには現在のCPU使用率、プロセッサ名、直近の変化が表示されます。RAMカードには使用率、使用量、総容量、空き容量が表示されます。

## 使用量の多いプロセスを見る

CPUまたはRAMカード右上の矢印を押すと、使用量の多いプロセス一覧を展開できます。もう一度押すと閉じます。

- CPUではプロセスごとのCPU使用率を表示します。
- RAMではプロセスごとのメモリ使用量を表示します。

値はシステム情報の更新とともに変化します。

## 詳細な履歴を見る

Footerのグラフアイコンを押すと、CPUとRAMの履歴画面が開きます。

- **Now**: 最新の値
- **Avg**: 表示期間内の平均値
- **Max**: 表示期間内の最大値
- **Last**: グラフが表している期間

表示期間はUpdate intervalによって変わります。
