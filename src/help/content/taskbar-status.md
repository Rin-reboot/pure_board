# タスクバーのミニグラフ

pure_boardは、Windows 11の通知領域にCPU、RAM、Networkの状態を示すミニグラフを表示します。ウィンドウを開いているときも、タスクトレイへ隠しているときも利用できます。

## Windows 11で常に表示する

Windows 11では、新しいアプリのアイコンが通知領域の「隠れているインジケーター」に収納されることがあります。pure_boardから常時表示を強制することはできないため、次のいずれかで表示してください。

1. タスクバー右端の **^** を押します。
2. pure_boardのアイコンを、音量やネットワークアイコンが並ぶ表示領域へドラッグします。

Windowsの **設定 > 個人用設定 > タスクバー > その他のシステム トレイ アイコン** からpure_boardを有効にする方法もあります。Windowsの更新により設定名が多少変わる場合があります。

## Settingsで変更できる項目

TitleBarの歯車からSettingsを開き、**Taskbar status**を変更します。

- **Show mini graph**: ミニグラフの有効・無効を切り替えます。初期値は有効です。
- **Metric**: `CPU`、`RAM`、`Network`から表示内容を選びます。
- **Update interval**: `1`～`60`秒を1秒単位で指定します。

アイコンへマウスを重ねるとCPU、RAM、Networkの現在値を確認できます。右クリックメニューの先頭にも、選択中の値が表示されます。

## 自動的に静止する条件

バッテリー駆動中、またはOSで「視覚効果を減らす」が有効な場合、電力消費と動きへの配慮のためミニグラフは最後の状態で静止します。現在値はツールチップと右クリックメニューで更新され、AC電源への接続または設定解除後にグラフ更新を再開します。

## Linuxでの違い

Linuxではデスクトップ環境とパネル実装により表示や操作が異なります。ミニグラフと右クリックメニューを基本動作とし、ツールチップや左クリックでの復元が利用できない環境があります。

この機能はWindows 11を優先して設計・検証しています。
