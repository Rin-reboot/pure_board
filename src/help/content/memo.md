# Memo

Memoはアプリを閉じた後も保存される簡易タスクリストです。

## メモを追加する

1. 入力欄の左でタグを選びます。
2. メモの内容を入力します。
3. Enterキー、送信アイコン、またはヘッダーの追加アイコンを押します。

空白だけのメモは追加されません。

## 完了・削除

- 左側のチェックボックスで完了状態を切り替えます。
- 右側の削除アイコンでメモを削除します。

## 一覧を絞り込む

ヘッダーのリストアイコンを押すと、拡張リスト表示になります。

- **すべて／未完了／完了** で状態を絞り込めます。
- タグ選択欄で特定のタグだけを表示できます。
- 左上の件数は「表示中 / 全件」を表します。

もう一度リストアイコンを押すと通常表示へ戻ります。
